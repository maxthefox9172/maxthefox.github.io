Linux troubleshooting

How do I play Neoalpha on Linux?

(Tested on Ubuntu 20.04 and Pop!_OS 20.04/22.04) In order to play Neoalpha on Linux you'll need to install the Zulu build of OpenJDK. OpenJDK itself won't launch Neoalpha. You can find these builds at this link. Note that the '.deb' file format can be installed only on Debian-based distributions (e.g. Ubuntu, Linux Mint, Pop!_OS) meanwhile '.rpm' file format can be installed only on Fedora Linux-based distributions (e.g. Red Hat Linux, Oracle Linux, CentOS).

I installed the Zulu build of OpenJDK but now I get stuck at a black screen. How do I fix that?

I'm still trying to figure out how to exactly fix this. You can try updating / installing graphics card drivers. Replacing the LWJGL files also supposedly fixes the issue (instructions on how to do that below). I can't guarantee that these solutions will work though.

How to replace the LWJGL files? * may not and probably won't fix the black screen issue, but you can try doing this

Go to the LWJGL 2 website (link here) and click 'Downloads' on the left side. Click the 'LWGJL 2.9.3' link. You will be redirected to SourceForge. Download 'lwjgl-2.9.3.zip'. Open the '.neoalpha' folder (should be located at /home/yourlinuxdistusernamehere/.neoalpha/. If you don't see it, you must click the 'Show Hidden Files' option.). In the '.bin' folder replace 'jinput.jar', 'lwjgl.jar' and 'lwjgl_util.jar' from the zip file (these files are located in the zip file at '/lwjgl2.9.3/jar'). In the 'natives' folder ('/home/yourlinuxdistusernamehere/.neoalpha/bin/natives/') replace all of the files with the files from the zip file (files from the zip are located at '/lwjgl2.9.3/native/linux').
